✦ What's inside

Everything you need
to write in the air.

🌟

Neon Glow Trails

Multi-layered bloom effect on every stroke — a wide outer halo and a bright white inner core.

💨

Fading Strokes

Drawings dissolve over ~3 seconds. Toggle fade on/off with F to keep strokes permanently.

🎨

Radial Colour Picker

Hold a fist to summon a circular palette of 8 colours around your wrist. Hover to select, open hand to dismiss.

🤌

Pinch-to-Resize

Pinch your thumb and index finger together or apart to dynamically resize your brush from 2px to 30px.

🖐️

Two-Hand Support

Right hand draws, left hand erases. Both tracked simultaneously for a natural, fluid experience.

📦

Zero Hardware

All you need is Python 3.10 and a webcam. Built on MediaPipe and OpenCV — both free and open source.

✦ Controls

All gestures.

☝️ Index finger up Draw on the canvas Draw
🤌 Pinch spread Adjust brush size (2–30px) Brush v3
✌️ Two fingers up Hover without drawing — pen lift Pause
✊ Fist (hold 0.7s) Open the radial colour picker Colour
🖐️ Open palm Erase strokes under your hand Erase
⌨️ Z key Undo the last stroke Undo v3
⌨️ S key Save canvas as PNG to ~/air_writer_saves/ Save v3
⌨️ F key Toggle stroke fade on / off Fade v3
⌨️ B key Toggle dark / light background BG v3
⌨️ C key Clear the entire canvas Clear

✦ Setup

Up in 3 steps.

Requires Python 3.8–3.11. MediaPipe doesn't support 3.12+.

# 1. Clone & enter
git clone https://github.com/veer-rohra/air-writer.git
cd air-writer
# 2. Create environment & install
conda create -n airwriter python=3.10 -y
conda activate airwriter
pip install opencv-python mediapipe==0.10.9 numpy
# 3. Run v3
python docs/air_writer_v3.py
